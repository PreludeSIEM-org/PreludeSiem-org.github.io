Description

Some rulesets have data in the log samples that could be used to populate classification.reference() fields. Almost all rulesets contain data in the log samples to populate additional_data() fields, which may be useful for correlation or query reasons.
Where available, all rulesets need to be extended with this information. The formatting of these fields needs to be consistent across the additional_data() fields (in particular, the type field should be abstract enough to allow correlation, but specific enough to be meaningful).
